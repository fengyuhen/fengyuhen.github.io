车道线检测资料

Key Point

  1. 校正图片
  2. 图片处理，得到车道线的二进制图
  3. 透视变换得到二进制图的鸟瞰图(birds-eye view)
  4. 检测车道线的像素，并用它来测出车道边界
  5. 计算车道的曲率，及车辆相对车道中央的位置
  6. 处理图片，展示

想法:

  • 优化车道线识别算法
  • 车辆高速运行对摄像机成像的影响（会轻微变形）

Reference:
